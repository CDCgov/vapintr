Welcome to vapintr!

The ATSDR Vapor Intrusion Simulation Tool, or vapintr for short, performs vapor intrusion simulations using the Johnson and Ettinger Model (JEM). The application can conduct both deterministic and stochastic simulations. Use the features on this screen to upload your data, view your inputs, and run the JEM. Once the model calculations are finished, the Results Summary screen provides the model results.

The Resources page contains examples of properly formatted data sets. For questions or assistance using vapintr, please contact the ATSDR technical team at PHAST@cdc.gov and include the phrase "vapintr" in the subject line.

Instructions: To run the JEM:

  1. Download the Excel data import template by clicking the "Download Data Import Template” link.
  2. Load your data into the template. Example files showing properly formatted data are available on the Resources page.
  3. Click the “Browse” button below and locate the Excel template with your data. Summary information about your simulation, including a table of simulation settings, a conceptual site model image, and a soil strata profiles image, will appear once the data have finished loading.
  4. Scroll down and click the "Run JEM Simulation" button to run the JEM using your simulation settings.
